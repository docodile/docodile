# Markdown

`gendoc` supports all basic Markdown syntax.

## Headings

```markdown
# Heading level 1

## Heading level 2

### Heading level 3

#### Heading level 4

##### Heading level 5

###### Heading level 6
```

## Paragraphs

## Emphasis

You can use _italics_ and **bold** to emphasise parts of text. You can combine **_bold and italics_**.

```markdown
You can use _italics_ and **bold** to emphasise parts of text. You can combine **_bold and italics_**.
```

